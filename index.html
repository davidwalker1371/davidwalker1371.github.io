HomeTask

HomeTask is an application that tracks recurring home maintenance items (or any other recurring maintenance). It allows any task frequency and alerts the user when they are due, while tracking completions. Alerts can be configured for badges (red number appearing on the app icon), alerts, and sound. Tasks can be marked for completion, where they are assigned and new due date and completion date recorded. All data is kept in iCloud, which can be shared on different devices using the same iCloud user. Dark mode is supported.

Poker Trainer

Poker Trainer is an application that allows the player to achieve the best possible return from each hand. It provides a detailed analysis of how this is determined. Two separate responses are provided for games where higher returns are provided when the max bet is selected (e.g. Jacks or Better Royal Flush pays 250 to 1 for 1-4 coins and 800 to 1 for 5 coins). Games supported include Jacks or Better, Deuces Wild, Double Bonus, Double Double, and Triple Double.

CarTask

CarTask is an application that tracks automobile maintenance activities. All data is kept in iCloud, which can be shared on different devices using the same iCloud user. VIN numbers can be scanned using your car's barcode.

Loan Calculator

Loan Calculator is an easy simple interest loan payment calculator. Also shows loan payment, balance, and interest history. Dark mode is supported.

MyLists

MyLists is a simple list manager. It supports multiple lists while separating completed items. All list information is kept in the Cloud. Dark mode is supported.

Contractor Time Clock

Contractor Time Clock is an application that tracks time spent on jobs. Timesheets and invoices can be generated and shared with other users via Text or Email. Multiple jobs can be used, allow time and rates to be differentiated. All data is kept in iCloud, which can be shared on different devices using the same iCloud user.
